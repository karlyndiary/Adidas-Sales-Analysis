- Operating Profit Margin (%)
```
=AVERAGE(sales[operating_margin])
```
- Total Operating Profit (in Millions)
```
=SUM(sales[operating_profit]) / 1000000
```
- Average Revenue Growth Rate
```
=AVERAGE(sales!R2:R9649)
```
- Return on Sales (ROS)
```
=AVERAGE(sales!L2:L9649)
```
